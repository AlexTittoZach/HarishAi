# Contributing to HarishAI

Thank you for your interest in contributing to HarishAI! We welcome contributions from everyone who wants to help make mental health support more accessible.

### Reporting Issues
- Use the GitHub issue tracker to report bugs
- Include detailed steps to reproduce the issue
- Provide information about your environment (OS, browser, etc.)

### Suggesting Features
- Open an issue with the "feature request" label
- Describe the feature and its potential benefits
- Consider the impact on user privacy and mental health

### Code Contributions

1. **Fork the repository**
2. **Create a feature branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. **Make your changes**
   - Follow the existing code style
   - Add tests if applicable
   - Update documentation as needed
4. **Commit your changes**
   ```bash
   git commit -m "Add: brief description of your changes"
   ```
5. **Push to your fork**
   ```bash
   git push origin feature/your-feature-name
   ```
6. **Open a Pull Request**
